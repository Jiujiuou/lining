# 订单用户数据导出（独立扩展）

从千牛 / 淘宝「已卖出订单」分页拉取订单号与买家昵称，导出 CSV。

## 与「数据获取」主扩展同时安装

- **独立扩展**：本扩展使用独立协议（`OU_GET_USER_DATA`、`START_OU_USER_DATA`、`OU_USER_DATA_*`）；主扩展「数据获取」已不再包含订单用户导出，可仅安装本扩展完成导出。
- 在**同一页面**分别点击两个扩展的「获取用户数据」时，各自只处理自己的消息与下载；不会出现主扩展误下两份 CSV 的情况。
- 主世界会各注入一份脚本（不同全局变量 `__LINING_OU_USERDATA_MAIN__` / `__LINING_SOLD_USERDATA_MAIN__`），`fetch` 为链式包装，两路 `capturedBody` 互不覆盖。

## 加载方式

Chrome → 扩展程序 → 开发者模式 → 「加载已解压的扩展程序」→ 选择本目录 `src/extension-order-userdata`。

## 使用

1. 打开千牛已卖出订单或 `trade.taobao.com` 对应列表页并保持登录。
2. 点击扩展图标，填写可选筛选后点「获取用户数据」。
3. 完成后浏览器会下载 `sold_userdata_ou_*.csv`（与主扩展导出的文件名前缀不同，便于区分）。
